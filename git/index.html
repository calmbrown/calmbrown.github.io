    Git remote 설정 방법( + 서브 모듈-디렉토리 설정 )

    우선 origin으로 설정할 디렉토리를 만든다.

    참고 : 서브 디렉토리는 submodule 명령어 입력하면 자동으로 생긴다.

    cd [git 동기화할 디렉토리]
git init
git remote add orgin [orign repo url]   
git submodule add -b master [sub repo url] [sub directory]   
git add .   
git commit -m "example message"
git push origin master

    위에서 orign이 이미 있다고 에러가 날때, 아래 명령어로 origin을 제거하고 추가한다.

    git remote rm origin
